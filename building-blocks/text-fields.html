--- layout: default title: 텍스트 필드 ---

텍스트 필드는 여러분의 앱에 사용자가 텍스트를 입력할 수 있게합니다. 텍스트 필드는 한 줄 또는 여러 줄이 될 수 있습니다. 텍스트 필드를 터치하면 커서를 표시하고 자동적으로 키보드를 표시합니다. 텍스트 입력에 추가적으로 텍스트 선택(자르기, 복사하기, 붙여넣기)과 자동 완성을 통한 데이터 찾기 같은 다양한 기능이 있습니다.

한 줄과 여러 줄

한 줄 필드는 텍스트 입력 커서가 입력 필드 오른쪽 끝에 위치하면 자동적으로 내용을 왼쪽으로 스크롤 합니다. 여러 줄 텍스트 필드는 넘치는 텍스트를 자동적으로 새 줄로 나눠주고, 커서가 아래줄에 닿으면 자동적으로 세로로 스크롤합니다.

텍스트 필드 형태들

텍스트 필드는 숫자, 메세지, 이메일 주소와 같이 다른 형태들을 가집니다. 어떤 종류의 문자들을 필드에 입력하느냐에 따라 형태를 결정하고, 자주 사용하는 문자들 배열로 최적화된 가상 키보드를 표시합니다.

자동완성 텍스트 필드

실시간 완성 또는 검색결과를 팝업에 표시하는 자동완성 텍스트 필드를 사용하면 사용자는 더 정확하고 효율적으로 정보를 입력할 수 있습니다.

텍스트 선택

사용자는 롱 프레스로 텍스트 필드의 단어들을 선택할 수 있습니다. 이것은 선택을 확장하거나 선택된 텍스트에 수행할 작업을 고를 수 있게 해주는, 텍스트 선택 모드를 실행합니다. 선택 모드에 포함되는 것 :

  1. 컨텍스트 액션 바

    컨텍스트 액션 바(CAB)는 선택된 것에서 수행할 수 있는 작업들을 표시합니다.: 전형적으로 자르기, 복사하기 그리고 붙여넣기 기능이지만 앱들은 필요한 명령을 추가할 수 있습니다.

  2. 선택 조정

    선택 모드 상태에서 선택 조정은 텍스트를 더 많게 또는 적게 선택하도록 드래그할 수 있습니다.
