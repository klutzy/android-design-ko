안드로이드 디자인 언어는 스케일, 공백, 리듬, 기준선 격자와 같은 전통적인 타이포그래피 도구에 의존하고 있습니다. 이러한 타이포그래피 도구는 사용자가 빠르게 화면 상의 정보를 이해할 수 있도록 돕는데 매우 중요합니다. 이와 같은 타이포그래피를 지원하기 위해 아이스크림 샌드위치는 UI 요구사항과 고해상도 화면을 위해 특별히 고안된 로보토(Roboto)라 이름한 새로운 글꼴을 소개했습니다. 현재 TextView 프레임워크는 일반, 볼드, 이탤릭, 그리고 볼드 이탤릭을 기본으로 지원합니다.

기본 타입 색상

안드로이드 UI는 다음과 같은 기본 컬러 스타일을 사용합니다: textColorPrimary와 textColorSecondary. light 테마에서는 textColorPrimaryInverse와 textColorSecondaryInverse를 사용하세요. 이 프레임워크 텍스트 컬러 스타일은 UI 요소 안에서 사용될 경우 터치 피드백 상태 또한 지원합니다.

타이포그래피 크기

정상적인 레이아웃에서 글자 크기를 대비시키는 것은 정돈을 위해 도움이 될 수 있습니다. 그래도 같은 UI 내의 너무 다양한 글자 크기는 지저분하게 느껴질 수 있습니다. 따라서 안드로이드 프레임워크는 다음과 같은 제한된 글자 크기 세트를 사용하고 있습니다:

사용자는 설정 앱에서 시스템 전역에 적용되는 글꼴 확대 수준을 선택할 수 있습니다. 이러한 접근성 기능을 지원하려면 가능한 부분은 모두 스케일 독립 픽셀(sp) 타입으로 지정되어야 합니다. 확대 가능한 타입으로 지정된 레이아웃은 이러한 설정에 대한 테스트가 필요합니다.
